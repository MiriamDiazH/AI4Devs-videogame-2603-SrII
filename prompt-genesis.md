# Raw Project Context - Video Game Clone: Contra





## Project general idea



This is a video game clone, Contra. All the project should be at and only folder /Contra-AC. Don't edit or put anything outside that folder.



## Tech Stack



- Html

- JS using Phaser library

- Pure CSS



For development, use the skills `feature-dev:feature-dev`, `context7` and `simplifier`

The project resources are at /Contra-AC/resources.



## Game features



- Platform game.

- One single player.

- Player can shoot.

- There are enemies who appear at random times and approach the player.

- There is a countdown for the level.
